# TinyLink

## Overview
TinyLink is a URL shortening service designed to create shorter, manageable links from long URLs. It provides features like link tracking, analytics, and more.

## Features
- **Shortening URLs:** Convert long URLs into short ones.
- **Analytics:** Track how many times a link has been clicked.
- **Custom Links:** Option to create custom links for branding.
- **User Management:** Ability to manage your links through user accounts.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kr-sourav1/TinyLink.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TinyLink
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the server:
   ```bash
   npm start
   ```
2. Access the application in your browser at `http://localhost:3000`
3. Follow the UI to shorten your URLs.
